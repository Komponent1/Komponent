# Button

버튼(Button)은 유저의 액션에 반응하여 동작을 실행할 수 있습니다.

아래와 같이 사용합니다.

```tsx
import { Button } from '@seolim/react-ui';

const Component = () => (
  <Button>Button</Button>
);
```
