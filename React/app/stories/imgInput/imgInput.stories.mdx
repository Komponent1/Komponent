# ImgInput

이미지인풋(`<ImgInput/>`)은 폼(form) 요소 중 file입력(type="file")에서 이미지만 받는 요소로 집약한 UI입니다.

유저의 입력에 따라 데이터를 갱신하고 폼을 통해 데이터를 서브밋(submit)합니다.

모든 폼은 `useFormControl`을 통해 `FormControl`을 생성해 사용할 수 있습니다.

```typescript
import { useFormControl, ImgInput } from '@seolim/react-ui/form';

const Component = () => {
  const control = useFormControl<File, HTMLInputElement>({});
  return (
    <ImgInput control={control} id="unique_id"/>
  );
}
```

이미지인풋은 클릭을 통한 입력과 dragDrop을 통한 업로드를 동시에 제공합니다.

또한 이미지인풋은 기본적인 파일 타입검사, 파일용량검사를 지정한 값에 따라 검증하는 로직을 제공합니다.
