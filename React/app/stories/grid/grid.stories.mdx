# Grid

그리드(grid)는 UI를 배치하는 레이아웃 입니다. CSS grid와 유사하게 동작합니다.

```typescript
import { Grid } from '@seolim/react-ui';

const Component = () => {
  return (
    <Grid>
      {...content}
    </Grid>
  )
}

```

컬럼의 개수를 정할 수 있습니다. 컬럼은 부모 width를 기준으로 정방으로 나눠집니다.

<Canvas>
  <Story name="grid"
    parameters={{ layout: 'centered' }}>
    <Grid>
      <span>GRID1</span>

      <span>GRID4</span>
    </Grid>
  </Story>
</Canvas>

별도로 추가된 기능은 미디어 쿼리(media query)의 적용입니다. break포인트를 설정하면 해당 브레이크 포인트마다 컬럼의 개수가 감소합니다.

break포인트는 높은 해상도부터 작성합니다.

```typescript
const Component = () => (
  <Grid breakPoint={['1024', '564']} />
)
```
