# SRUI
<p style={{ fontSize: '16px' }}>SURI는 react project에서 사용할 수 있는 컴포넌트 디자인 시스템입니다.</p>
<p style={{ fontSize: '16px' }}>layout, component, form과 관련된 디자인 시스템을 제공합니다.</p>


## Installation & Use

<p style={{ fontSize: '16px' }}>SURI는<a href="https://www.npmjs.com/package/@seolim/react-ui">npm package</a>에서 확인할 수 있습니다.</p>
<p style={{ fontSize: '16px' }}>설치 전 사전적으로 설치되어야하는 package를 선행 설치해야합니다.<span style={{ color: 'red', fontSize: '14px' }}> react, react-dom 버전 16.8 이상에서만 동작합니다(hook 활용)</span></p>

```bash
# peer dependency
npm install @emotion/styled @emotion/react react react-dom react-bootstrap-icon
# installation
npm install @seolim/react-ui
```
