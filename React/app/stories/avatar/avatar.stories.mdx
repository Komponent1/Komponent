# Avatar

아바타(Avatar)는 개인(혹은 정보)을 표현하는 작은 뱃지와 같은 역할을 합니다.

아바타의 형태는 아미지미나 아이콘일 수 있고 문자일 수 도 있습니다.

```typescript
import { Avatar } from '@seolim/react-ui';

const Component = () => <Avatar src="" alt="" />
```

SRUI는 내부에 이미지, 컴포넌트, 텍스트를 표시할 수 있습니다.

이미지는 **src**, 컴포넌트와 텍스트는 children을 넘겨주면 됩니다.

SRUI의 아바타는 이미지의 로드 실패를 설정할 수 있습니다. alt를 설정하면 이미지 로드 중, 혹은 실패시 설정한 alt를 화면에 표시합니다.

우선 순위는 이미지 > alt > children 순서입니다.
