### 기능
1. 최초에 데이터를 받아와서 Tab을 생성
2. Tab은 각각의 nav를 클릭시 해당 요소의 context를 보여준다..

- 최초에 데이터를 받아온다.
- 상단에 Navigator가 있다.
- Nav는 받아온 데이터의 숫자만큼이다.
- Nav를 클릭하면 Nav에 해당하는 데이터를 화면에 표시한다
- 선택된 Nav는 선택이 되었음을 표현한다
