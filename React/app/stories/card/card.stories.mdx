# Card

카드는 기본적으로 데이터를 시각화 하는 도구로 통상 특정 정보의 디테일로 들어가는 진입점 역할을 합니다.

기본적으로 카드는 몇가지 요소로 구성으로 되어있는데, 필수적으로 헤더와 액션이 포함되고 부가적으로 미디어와 설명(데스크립션)이 포함됩니다. 디자인 구성이 아래와 같이 다양하게 나타날 수 있습니다.

<Story name="layout">
  <img src={'/card_show.png'} width="500px"/>
</Story>

이중 액션은 다양한 형태로 나타나는데, collapse(확장 UI), 공유, 아이콘, 버튼등등 다양한 형태로 나타납니다. 

```tsx
import { Card, CardHeader, CardMedia, CardContent } from '@seolim/react-ui/card';

const Component = () => (
  <Card>
    <CardHeader
      title="title"
      subtitle="subtitle"/>
    <CardMedia src="image src" />
    <CardContent>
      description of Card
    </CardContent>
  </Card>
);
```

카드는 `<Card>`, `<CardHeader>`, `<CardMedia>`, `<CardContent>` 의 조합을 통해 구현할 수 있습니다.

`<Card>`는 카드를 감싸는 레이아웃입니다.

`<CardHeader>`는 타이틀과 간략 설명을 포함합니다.

`<CardMedia>`는 카드에 적용될 이미지를 포함합니다.

`<CardContent>`는 카드의 설명, 옵션, 액션을 포함하는 바디입니다.

카드는 2가지 디자인과 액션에 대한 UX(floating)을 제공합니다. border는 액션없이, shadow는 액션을 포함하는것을 권장합니다.

### CardLoading

카드는 데이터 로딩에 따른 로딩 UI(`<CardLoading />`)를 별도로 제공합니다. 스켈레톤의 형태를 띄고 진행되며 카드와 동일한 형태를 띌 수 있습니다.

```typescript
import { CardLoading, Card } from '@seolim/react-ui/card';

const Component = () => {
  const { data, loading } = useFetch('api');

  return (
    <>
      {(loading || !data)
        ? <CardLoading />
        : <Card />}
    </>
  )
}
```

로딩카드는 본인이 가질 로딩 요소를 선택할 수 있습니다. `content` 프로퍼티를 통해 전달합니다.
