# Skeleton

스켈레톤(Skeleton)은 특정 UI가 로드되고 있음을 알리고 어떠한 형태로 화면에 표시될지를 보여주는 UI입니다.

스켈레톤은 높이(height)와 너비(height)의 설정을 권장합니다. 스켈레톤은 로드될 화면의 크기를 알 수 없습니다.

```typescript
import { Skeleton } from '@seolim/react-ui/loading';

const Component = () => <Skeleton width={100} height={100} />;
```

SRUI의 스켈레톤은 세가지 형태를 제공합니다.
