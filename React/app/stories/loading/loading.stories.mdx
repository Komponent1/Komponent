# Loading

로딩(Loading)은 유저가 요청한 동작이 진행되고 있음을 알려주는 UI입니다.

SRUI는 3가지 로딩 타입을 제공합니다.

모든 로딩은 2가지 크기(scale)와 속도(speed)를 제공합니다.

### 점(Dots)

```typescript
import { Dots } from '@seolim/react-ui/loading';

const Component = () => <Dots />;
```
